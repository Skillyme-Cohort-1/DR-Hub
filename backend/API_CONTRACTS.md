## Booking A SLOT => David
- For a user to make a booking;-
    - If they do not have an account, create for them.
    - If they have an account, Associate the booking to that.
    - A booking should only happen on an available slot, i.e, booked=false
    - A user should only book a slot on a room for a certain date not the room.
    - After booking is complete, send the user a notification, i.e email/sms/whatsapp with instruction to pay.
    - By default a booking should always be in Pending status.
